# notepad-test

 
Consiste en automatizar los siguientes escenarios. Se requiere mínimo
automatizar 1 (cualquiera), se obtiene un mejor puntaje al lograr la
automatización de todos los escenarios:

Scenario 1: Crear una nota de texto enriquecido con letra en negrita

Scenario 2: Eliminar un artículo de una lista de 5 elementos

Scenario 3: Mover un artículo de posición de una lista de 5 elementos

- Cada escenario debe terminar de definirse y automatizarse usando
Screenplay.
- Url de la página a automatizar:
https://www.online-notepad.net/
- Deberá modificar el arquetipo en caso de verlo necesario, depurar
información que no sea relevante y cumplir con las buenas prácticas de
nombramiento y demás.
